# 在原作者基础上做了 ESP-01s模块的适应
**1、3开关模式的Tamplate.h, IO的调整，IO0控制继电器，IO2控制继电器小板LED  **
**2、开关高低电平调整，由高电平打开继电器改为低电平打开继电器  **
**3、增加功能，将语言的turn on指令映射为toggle，即打开指令变为交替打开和关闭  **
